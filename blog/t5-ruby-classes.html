Ruby Classes

3 - 08 - 2015

Classes are layouts that contain the structure of Objects, their behavior and attributes. We use Classes to model real life objects and create instance of them. To better explain Classes we will create one, that model a Person.
So let’s think what attributes a person has: Name, Last Name, Gender, Day of Birth And any other that you could think of, a Class can have as many attributes as you need. Now let’s think of what a person do, and we could use this as methods to simulate their behavior. Walk, Jump, Eat, Sleep, Talk, Breath And so on…
Our class will look like this 

							
				class Person
					def initialize (name,last_name,gender,dob)
						@name = name 
						@last_name = last_name
						@gender = gender
						@day_of_birth = dob
					end

					def walk
						#walking, we could define how fast a person walks or for how long
					end

					def jump
						#jumping, define how high and how long 
					end

					def eat
						#eating, what type of food, how many, how he eats
					end

					def sleep
						#sleeping, how many hours, on what side
					end

					def speak
						#speaking, in what language, how fast
					end

					def breath
						#breathing, how often, how it breath
					end

				end
Notice the initialize method? In this method we initializing every attribute for this person, giving it a name, last name a gender and a day of birth. When creating an instance of a class Person we are creating an object of type Person that will have it owns attributes and behavior, so when doing : 

Person.new(Luis,Echenique,male,09-11-1980)
we are simulating a person (object) , with this attributes.
