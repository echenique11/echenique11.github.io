What is Rails?

03-29-2015

Rails is a web application development framework written in the Ruby language. As all frameworks are design to make development easier and faster, providing built in functionality that will make your life as a programmer easier.
It implements a MVC pattern, Model View Controller, which separate functionality in different layers. Accessing data, showing the data and business rules controllers.
In rails you will find lots of libraries that will abstract you from connections to database, session manager, security controls and helpers that will save you time when coding. This is very helpful since all those libraries and helpers has been implemented, using what Rails community understand is the best way to do things, so you will get a better performance for your webapp and also will help you to have less bugs in your code.
