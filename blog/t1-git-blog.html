What are the benefits of version control?

To know the benefits of version control, you need to know first what version control is, right?. Version control is a system that keeps track of all changes made to a file/folder and allows you to go back in history of that file/folder to get a specific saved version. Knowing that, some of the version control benefits are :

  • Retrieve old versions of a file/folder
  • Display changes made to a file/folder
  • Work with an updated version of the file/folder
  • Create new version of the project

How does git help you keep track of changes?

git is a version system that store all changes made to a file as a different version of the file, and allows you to check the history of the file and merge with others version to update the file.

Why use GitHub to store your code?

GitHub is a social site to share projects, here you can push or pull your code or someone's else.
